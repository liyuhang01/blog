## 安装next主题

```
git clone https://github.com/theme-next/hexo-theme-next themes/next
```

## 安装插件

```
npm install
```
## 对站点文件进行一些配置

- 站点配置文件下：
  1、你的网站信息
  2、你的deploy
  3、baidu_url_submit
- next.yml配置文件中：
  1、social
  2、github_banner
  3、SEO设置 
  4、评论设置

- 当然，你也可以直接删除掉你不需要的配置，比如你没有SEO相关的设置，你直接在next.yml中删除掉就好

## 开始使用

```
hexo g
hexo s
```
